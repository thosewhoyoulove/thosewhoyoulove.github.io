# 高阶加分点（展示技术深度）

面试或述职时可在合适环节**点到为止**地提及，展示你不只停留在「会用 API」。每项都可再单独扩成一篇笔记。

## 安全维度

- **HTTPS 证书链**：服务端链、中间证书、OCSP/CRL；CT 日志；HPKP 已废弃，了解即可。
- **跨域与 Cookie**：CORS 预检、**SameSite**（Lax/Strict/None）与第三方 Cookie 策略演变。
- **浏览器沙箱**：站点隔离（Site Isolation）、进程模型与 Spectre 类侧信道背景（不必深入 CPU）。

## 性能优化

- **关键渲染路径（CRP）**：与 [浏览器的渲染原理](/md/浏览器/浏览器的渲染原理.md) 串联。
- **资源提示**：`preload` / `prefetch` / `preconnect` / `dns-prefetch` 适用场景与滥用代价。
- **HTTP/2 Server Push**：已式微，可对比现代 **103 Early Hints** 或 HTTP/3 场景。

## 现代浏览器特性

- **多进程架构**：Browser / GPU / Network / Renderer 等分工（不同浏览器命名略有差异）。
- **V8**：Ignition + TurboFan 管线大意（字节码解释 + 优化编译），不必背细节。
- **存储与隐私**：`SameSite`、`Storage Access API`、分区存储（第三方 iframe）等趋势。

## 网络协议演进

- **QUIC / HTTP3**：基于 UDP 的可靠传输、0-RTT 取舍、连接迁移、队头阻塞改善点。
- **WebTransport**：低延迟双向流，与 WebSocket 对比（仍在生态发展中，答「了解场景」即可）。
