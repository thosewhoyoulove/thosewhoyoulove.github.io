# omen's blog

个人前端学习与面试笔记，基于 [Docsify](https://docsify.js.org/) 搭建，部署在 GitHub Pages。

## 如何使用本站

- 左侧目录按 **基础 → 网络/浏览器/安全 → 工程化 → 框架** 组织，可按主题系统阅读。
- 顶部搜索框可全文检索（标题与正文）。
- **面试准备** 下分 **技术**、**项目与架构**、**综合**：多为高频问题 + 答题要点，可对照左侧专题文深化理解。
