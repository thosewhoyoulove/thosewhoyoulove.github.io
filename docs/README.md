# omen's interview notes

个人前端面试知识库，基于 [Docsify](https://docsify.js.org/) 搭建，部署在 GitHub Pages。

## 面试优先入口

如果近期要面试，建议直接从这几篇开始：

- [面试总览](/md/面试准备/面试总览.md)：了解整站复习方式和知识地图。
- [复习路线](/md/面试准备/复习路线.md)：按三天冲刺或一周系统复习安排阅读顺序。
- [高频追问清单](/md/面试准备/高频追问清单.md)：用于面试前自测。
- [项目表达模板](/md/面试准备/项目表达模板.md)：把项目经历整理成能被追问的回答。

左侧目录已经按面试场景重组：**面试总览 → JavaScript 与浏览器 → 网络与安全 → 框架 → 工程化与性能 → 项目与架构 → 协作与加分项**。

## 答题方式

每个知识点建议按这套结构准备：

1. 一句话结论。
2. 核心原理。
3. 项目中的使用场景。
4. 方案取舍和边界。
5. 如何验证结果。
