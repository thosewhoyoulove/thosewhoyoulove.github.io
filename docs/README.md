# omen's interview notes

个人前端面试知识库，基于 [Docsify](https://docsify.js.org/) 搭建，部署在 GitHub Pages。

左侧目录按技术方向组织：**JavaScript 与浏览器 → 网络与安全 → 框架 → 工程化与性能 → 项目与架构 → 协作与加分项**。

## 答题方式

每个知识点建议按这套结构准备：

1. 一句话结论。
2. 核心原理。
3. 项目中的使用场景。
4. 方案取舍和边界。
5. 如何验证结果。
